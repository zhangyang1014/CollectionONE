# 甲方字段展示配置 - 快速使用指南

## 🚀 快速开始

### 访问页面
```
前端URL: http://localhost:5173/field-config/display
菜单路径: 字段配置 > 甲方字段展示配置
```

## 📋 主要功能

### 1️⃣ 选择场景

在页面顶部选择要配置的场景：
- **控台案件管理列表**: 管理后台使用
- **催员案件列表**: 催员端列表页
- **催员案件详情**: 催员端详情页

### 2️⃣ 添加字段配置

点击"添加字段配置"按钮，按以下步骤操作：

#### 第一步：选择字段
- 从下拉列表中选择字段
- 字段按分组展示（贷款详情、客户基础信息、借款记录等）
- 显示字段类型（String/Integer/Enum等）
- 显示字段来源（标准/扩展/自定义）

#### 第二步：基本配置
- **字段标识**: 自动填充，不可修改
- **字段名称**: 自动填充，可以修改显示名称
- **字段类型**: 自动填充，只读
- **字段来源**: 自动填充，只读
- **排序顺序**: 设置字段显示顺序，数字越小越靠前
- **显示宽度**: 设置列宽（像素），**填写0表示自动**
- **颜色类型**: 选择普通/红色/黄色/绿色

#### 第三步：搜索筛选（可选）
- **是否可搜索**: 针对文本字段，开启后可在列表中搜索
- **是否可筛选**: 针对枚举字段，开启后可在列表中筛选
- 系统会根据字段类型自动推荐设置

#### 第四步：隐藏规则（仅催员端，可选）
- **对队列隐藏**: 选择要隐藏该字段的队列
- **对机构隐藏**: 选择要隐藏该字段的机构
- **对小组隐藏**: 选择要隐藏该字段的小组

### 3️⃣ 快速编辑

在列表中可以直接修改：
- **显示宽度**: 直接输入数字（0=自动）
- **颜色类型**: 下拉选择
- **可搜索**: 开关切换
- **可筛选**: 开关切换

修改后点击"批量保存"保存更改。

### 4️⃣ 拖拽排序

1. 鼠标悬停在拖拽图标上
2. 按住鼠标左键拖动行
3. 拖动到目标位置后松开
4. 点击"批量保存"保存新顺序

### 5️⃣ 编辑配置

1. 点击行尾的"编辑"按钮
2. 在对话框中修改配置
3. 点击"确定"保存

### 6️⃣ 删除配置

1. 点击行尾的"删除"按钮
2. 确认删除
3. 配置被删除

### 7️⃣ 复制场景配置

快速复制配置到其他场景：

1. 点击"复制场景配置"按钮
2. 选择源场景（当前场景）
3. 选择目标场景
4. 点击"确定"
5. 系统会将源场景的所有配置复制到目标场景

## 💡 使用技巧

### 字段来源说明
- 🟢 **标准字段**: 系统预定义，所有甲方通用
- 🟠 **扩展字段**: 基于标准字段扩展，有独立别名
- ⚪ **自定义字段**: 甲方自定义的特殊字段

### 显示宽度建议
- **0**: 自动宽度（推荐）
- **80-100**: 短字段（状态、数字等）
- **120-150**: 中等字段（姓名、编号等）
- **160-200**: 长字段（地址、描述等）
- **200+**: 超长字段（备注等）

### 颜色使用建议
- **普通**: 默认颜色
- **红色**: 警告信息（如：逾期天数、欠款金额）
- **黄色**: 提醒信息（如：即将到期）
- **绿色**: 正常信息（如：已还清）

### 搜索筛选建议

**适合搜索的字段**（文本类型）：
- 案件编号
- 客户姓名
- 手机号码
- 身份证号
- 贷款编号

**适合筛选的字段**（枚举类型）：
- 案件状态
- 产品类别
- 性别
- 教育程度
- 工作状态

### 隐藏规则使用场景

1. **敏感信息保护**
   - 对部分队列隐藏身份证号、收入等敏感字段
   
2. **差异化展示**
   - 不同机构展示不同字段
   - 不同小组关注不同信息

3. **权限控制**
   - 通过隐藏规则实现字段级权限控制

## ⚠️ 注意事项

1. **字段唯一性**
   - 同一场景下，同一字段只能配置一次
   - 如果字段已存在，添加时会提示

2. **批量保存**
   - 拖拽排序后记得点击"批量保存"
   - 快速编辑多个字段后，建议批量保存

3. **隐藏规则**
   - 隐藏规则仅对催员端生效
   - 管理后台不受隐藏规则影响

4. **场景独立**
   - 3个场景的配置是独立的
   - 修改一个场景不影响其他场景

## 🎯 常见场景配置示例

### 场景1：控台案件管理列表
适合管理员查看全面的案件信息

推荐配置字段：
1. 案件编号（宽度：180，可搜索）
2. 贷款编号（宽度：150，可搜索）
3. 客户姓名（宽度：120，可搜索）
4. 手机号码（宽度：140，可搜索）
5. 案件状态（宽度：120，可筛选）
6. 逾期天数（宽度：100，颜色：红色）
7. 应还未还金额（宽度：130，颜色：红色）
8. 所属队列（宽度：100）
9. 催员姓名（宽度：100）
10. 分配时间（宽度：160）

### 场景2：催员案件列表
适合催员快速查看待处理案件

推荐配置字段：
1. 案件编号（宽度：150，可搜索）
2. 客户姓名（宽度：100，可搜索）
3. 手机号码（宽度：130，可搜索）
4. 逾期天数（宽度：80，颜色：红色）
5. 欠款金额（宽度：100，颜色：红色）
6. 案件状态（宽度：100，可筛选）
7. 最后联系时间（宽度：140）
8. 联系次数（宽度：80）
9. 应还日期（宽度：120）

### 场景3：催员案件详情
适合催员查看单个案件的详细信息

推荐配置字段：
- 所有关键字段
- 按分组展示（基础信息、贷款信息、催收信息等）
- 设置合理的隐藏规则保护敏感信息
