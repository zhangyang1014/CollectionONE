# 案件列表查看催记功能 - 快速开始

## 功能位置

**路径**：案件管理 → 案件列表 → 操作列 → 查看催记

## 快速体验

### 步骤 1：访问案件列表
```
http://localhost:5174/cases
```

### 步骤 2：选择甲方
在页面顶部的甲方选择器中选择一个甲方（如：百通企业）

### 步骤 3：查看催记
在任意案件行的操作列中，点击"查看催记"按钮

### 步骤 4：使用筛选功能
弹窗打开后，可以：
- 在搜索框中输入案件ID进行搜索
- 使用触达人筛选器选择催员
- 使用触达渠道筛选器选择渠道
- 使用状态筛选器选择状态
- 使用结果筛选器选择结果
- 使用日期范围选择器选择时间范围

## 操作示例

### 示例 1：查看所有催记
1. 点击"查看催记"按钮
2. 直接查看列表中的所有催记

### 示例 2：按触达人筛选
1. 点击"查看催记"按钮
2. 在触达人下拉框中选择"张三"
3. 列表自动更新，只显示张三的催记

### 示例 3：按日期范围筛选
1. 点击"查看催记"按钮
2. 点击日期范围选择器
3. 选择开始日期和结束日期
4. 列表自动更新，只显示该时间范围内的催记

### 示例 4：组合筛选
1. 点击"查看催记"按钮
2. 选择触达人："张三"
3. 选择渠道："WhatsApp"
4. 选择状态："可联"
5. 列表自动更新，显示符合所有条件的催记

## 常见问题

### Q1: 为什么每个案件的催记都一样？
A: 当前使用的是 Mock 数据，实际部署时会连接后端 API 获取真实数据。

### Q2: 筛选器不生效？
A: 请检查是否有数据匹配筛选条件。可以尝试清空筛选条件重新选择。

### Q3: 日期筛选如何使用？
A: 点击日期范围选择器，先选择开始日期，再选择结束日期，系统会自动筛选该时间范围内的催记。

### Q4: 备注内容显示不全？
A: 将鼠标悬停在备注单元格上，会显示完整的备注内容。

### Q5: 如何清除筛选条件？
A: 点击每个筛选器的清除按钮（×图标）即可清除该筛选条件。
