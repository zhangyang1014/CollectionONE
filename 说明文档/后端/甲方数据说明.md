## 案件数据特点

### 状态分布
- **进行中**: 未到期或刚到期的案件
- **逾期**: 已逾期1天以上的案件
- **已结清**: 约15-20%的案件已完成还款

### 逾期天数分布
- -7天 到 +60天（BTQ）
- -7天 到 +45天（BTSK）
- 颜色标识：
  - 绿色：负数（未到期）
  - 橙色：0天（当日）
  - 红色：正数（已逾期）

### 金额范围
- **BTQ**: 500-10,000 MXN
- **BTSK**: 6,000-120,000 INR（分期总额）

### 应答渠道
- 数值范围: 0-5
- 表示本人通过电话、IM等渠道应答的次数

## 数据生成脚本

数据由以下脚本生成：
- `generate_mock_cases.py` - 生成案件数据
- `custom_fields_config.json` - 自定义字段配置
- `generated_cases.json` - 生成的案件数据文件

如需重新生成数据，运行：
```bash
cd backend
python3 generate_mock_cases.py
```

## 注意事项

1. **单期 vs 多期**:
   - BTQ采用单期模式，到期一次性还款
   - BTSK采用多期模式，分期按月还款

2. **货币单位**:
   - 显示金额时注意区分MXN和INR
   - 前端可根据甲方的currency_code自动显示相应货币符号

3. **时区**:
   - 墨西哥: America/Mexico_City (UTC-6)
   - 印度: Asia/Kolkata (UTC+5:30)

4. **自定义字段**:
   - 每个甲方的自定义字段不同
   - 前端需要动态渲染这些字段
   - 字段类型包括：text, number, select
