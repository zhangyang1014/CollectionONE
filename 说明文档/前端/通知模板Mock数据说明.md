# 通知模板 Mock 数据说明

## 概述

已创建10条通知模板Mock数据，用于测试和演示通知模板功能。

## 📦 Mock数据文件

**文件位置**：`backend/mock_notification_templates.sql`

## 🚀 使用方法

### 方法1：MySQL命令行

```bash
cd backend
mysql -u root -p cco_system < mock_notification_templates.sql
```

### 方法2：MySQL客户端工具

1. 打开Navicat/MySQL Workbench等工具
2. 连接到数据库
3. 选择 `cco_system` 数据库
4. 打开并执行 `mock_notification_templates.sql` 文件

### 方法3：直接复制执行

复制SQL文件内容到MySQL客户端执行。
